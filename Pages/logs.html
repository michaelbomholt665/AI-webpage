Import Chat History

Command to Export Compatible Chat History

Generate a JSON file for our chat history in the following format: 
{
  "messages": [
    {
      "sender": "user" | "ai",
      "timestamp": "YYYY-MM-DDTHH:MM:SSZ",
      "content": "Message text"
    }
  ]
}
Ensure the following:
- The top-level key must be "messages".
- The "sender" field must be "user" or "ai".
- The "timestamp" field must be in ISO 8601 format (e.g., "2025-03-31T10:00:00Z").
- The "content" field must contain properly escaped text.
- Include all messages from our conversation.
- Ensure the JSON is syntactically valid.
